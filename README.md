# Data Analysis Portfolio
This repository showcases my data analysis projects as part of my education, career, or exploration. I utilize my skills in **Python (Pandas, Matplotlib)** and **RStudio (tidyverse, ggplot2)** to perform statistical analysis and data visualization on a range of data. 

## Projects

### Chicago Food Inspections Data Visualization and Analysis 
Cleaned and analyzed 310,000+ food inspection records from the City of Chicago Data Portal (2010–2026) to investigate whether high-risk geographic clusters improve over time. Aggregated data by zip code and risk level to engineer violation and failure metrics, then built interactive choropleth maps and time-series plots to identify persistent hotspots and temporal trends.

Skills used:
- Data cleaning and imputation (pandas, GeoPandas)
- Feature engineering and aggregation
- Geospatial and time-series visualization (Plotly)
- Statistical correlation analysis

Repository: https://github.com/ellis-jace/Food-Inspection-Data-Analysis

### Student Dropout Risk Analysis in Higher Education
Applied survival analysis methods to 4,448 university student records to identify which variables (gender, course of study, and first-semester academic performance (CFUP)) predict student dropout timing. Progressed from Kaplan-Meier estimation and log-rank testing through Cox Proportional Hazards modeling, diagnosing and resolving a proportional hazards violation with a time-varying coefficient model.

Skills used:

- Survival analysis (Kaplan-Meier, Cox PH, frailty models)
- Assumption testing (Schoenfeld residuals)
- Time-varying coefficient modeling (R, survival package)
- Statistical interpretation and reporting

Repository: https://github.com/ellis-jace/University-Dropout-Risk-Analysis
